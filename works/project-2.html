Web App

人才評價

為企業打造跨平台的人才評估系統，涵蓋 Web 管理後台與 App 端的受試者介面，提供完整的測驗流程體驗。

專案封面圖

專案背景

客戶需要一套數位化的人才評估解決方案，取代傳統紙本測驗。系統需同時支援管理者在 Web 端建立與管理評測，以及受試者在 App 端完成作答。

我的角色

負責 Web 與 App 雙端的 UX 流程設計與 UI 視覺設計。從需求訪談、競品分析到原型測試，完整參與設計流程。

設計細節展示

設計挑戰

最大的挑戰在於如何讓複雜的評測設定流程變得直覺易用，同時確保 App 端的作答體驗流暢，降低受試者的認知負擔。

成果

系統上線後受試者完成率提升至 92%，管理端的評測建立時間從平均 30 分鐘縮短至 10 分鐘以內。
